# 📚 지식나눔터 (corca-ai/library)

해외의 좋은 정보를 **한국어로 번역**해 함께 보는 공개 아카이브입니다.

## 🌐 바로 보기
👉 **https://corca-ai.github.io/library/**
모든 번역본이 날짜별(최신순)로 정리되어 있고, 클릭하면 바로 읽을 수 있습니다.

## 🗂 어떻게 정리되나
- 폴더 = **올린 날짜(한국 시간) `YYMMDD`**
  예) 2026-06-27에 올리면 → `260627/` 폴더
- 각 날짜 폴더 안에 그날 올린 번역 HTML이 들어갑니다.
- 메인 화면(`index.html`)은 자동으로 전체 목록을 만들어 줍니다.

```
library/
├── index.html      ← 자동 생성 메인 목록 (지식나눔터)
├── 260626/
│   └── (번역).html
├── 260627/
│   └── ...
└── ...
```

## ✍️ 어떻게 추가하나 (현재: 수동)
1. 번역 HTML 파일을 이 저장소 폴더 **맨 위(루트)** 에 둔다.
2. Claude에게 "올려줘"라고 말한다.
3. Claude가 → 오늘 날짜(KST) 폴더 생성 → 파일 이동 → 목록 갱신 → GitHub에 push.
   (직접 하려면: `./publish.sh`)

## 🛣 앞으로
지금은 손으로 올리지만, 이후 **번역→업로드 자동화 툴**로 확장할 예정입니다.
